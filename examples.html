Real Medical Billing Dispute Examples

Duplicate Charges Dispute

Patient billed twice for same X-ray (CPT 71046). Disputed $450 duplicate charge with specific CPT code reference.

Key Elements:

  • Specific CPT code (71046)
  • Date of service
  • Exact duplicate amount ($450)
  • Request for itemized bill

Result: $450 removed from bill

Balance Billing (Emergency Room)

Out-of-network ER doctor at in-network hospital. Disputed $3,200 balance bill citing No Surprises Act.

Key Elements:

  • No Surprises Act citation (Public Law 116-260)
  • 42 USC § 300gg-111 reference
  • Emergency service documentation
  • In-network facility proof

Result: $3,200 balance bill removed

Upcoding Dispute

Billed for Level 5 office visit (99215) when it was Level 3 (99213). Disputed $180 overcharge with time documentation.

Key Elements:

  • CPT 99215 vs 99213 comparison
  • Appointment duration (15 minutes)
  • CPT code definitions
  • Correct code calculation

Result: $180 adjustment applied
